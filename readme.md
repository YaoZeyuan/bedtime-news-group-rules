# Bedtime News Discussion Group Rules (Unofficial)

> 一个用于记录与维护“睡前消息讨论群”（非官方）群规及其变更历史的仓库。

---

## 📖 群组简介

本群为“睡前消息”观众自发组建的**非官方**讨论群，旨在通过理性讨论、资料共享与逻辑推演，理解现实运行逻辑，提升决策质量，并在长期视野下优化个人认知与行动策略（例如通过权益市场实现认知变现）。

群聊管理采用**结果导向的独裁模式**，目标是为成员提供**每日值得通读**的高质量讨论内容。

## 📜 群规概览

- [群规全文](./rule.md)
- [变更记录](./changelog.md)

### 1. 基本规则
- **独裁管理**：群主及指定管理员拥有最终解释与执行权。
- **高质量导向**：所有讨论应以事实、逻辑与证据为基础，追求信息增量与认知提升。
- **编辑保护**：群内存在编辑本尊（@豆农 / @穆好古 / @吉翁舰长）。**@编辑前必须向被@者发送10元定向红包**，否则将依规处理。

### 2. 违规处理机制
- **初次违规**：60分钟内补发红包，否则移出群聊一次（可扫码重进）。
- **三次违规**：禁言一周。
- **警告系统**：警告默认有效期为24小时，可明确告知后延长（最长不超过7天）。违规需@当事人并明确计数，**累计3次警告后移出群聊**。
- **重新入群**：被移出后可在 **1天 / 3天 / 7天** 后由任意群友拉回。

### 3. AI 辅助裁定
- 任何成员可随时使用主流AI工具（如 ChatGPT/DeepSeek）对发言进行**中性提问**，判断是否违规。
- 裁定后应在群内**发布判定链接**，并@管理员执行。
- 如多个AI结果冲突，以**对被判定方最有利的结果为准**。

### 4. 讨论共识（底线规则）
- ✅ 评价须基于：数据可靠性、证据链完整性、逻辑自洽性。
- ✅ 以**九年义务教育知识体系**为基础，以主流AI工具为中立参照。
- ✅ 发表争议观点需提供**可验证论据或权威文献**，拒绝“相对主义”话术。
- ❌ 禁止以下行为：
  - 以身份、立场、机构等非学术因素作为批判依据；
  - 用“祛魅学术界”等话术规避举证；
  - 拒绝AI校准基础概念并强行争辩定义。

## ⚖️ 执行细则与判例

### 避免 Troll 与低质发言
- 允许“废话率”低于50%，但长期只发链接/文宣类内容将视为 Troll，予以警告。
- 解除方法：增加实质性讨论，将废话率稀释至50%以下。

### 邀请连坐制
- 被邀请人入群**7天内违规**，累计计入邀请人次数。
- 如需规避，可提前“静默7天”再入群。

### 避免空谈
- 若连续三句发言短于15字且无论据，管理员可要求当事人在后续15分钟内每次发言**不少于50字**，包含论点与论据。

### 内容安全与氛围（参考厦门新城际规则）
- 不煽动情绪、不人身攻击、不暴露隐私、不强拉对立、不质疑动机。
- 提倡就事论事、尊重差异、语言诚恳直接，避免反讽与高级黑。

### 乐子内容边界
- ✅ 可接受：不针对特定群体、无争议、纯粹轻松幽默。
- ❌ 不可接受：笑点建立在攻击/贬低之上，或干扰严肃讨论节奏。
